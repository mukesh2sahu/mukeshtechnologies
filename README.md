# Mukesh Technologies

Welcome to the official repository for **Mukesh Technologies**, a premier digital solutions provider. We specialize in high-impact software, mobile apps, and stunning UI/UX designs.

## 🚀 Services

- **Software Development**: Building robust, scalable, and modern enterprise software.
- **Mobile App Development**: Crafting seamless iOS and Android experiences.
- **UI/UX Design**: Creating intuitive and visually striking user interfaces.

## 🛠 Tech Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com) (v4)
- **Font**: [Inter](https://rsms.me/inter/)
- **Visuals**: Premium 3D Rendered Assets

## 🏃 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **View Site**:
   Open [http://localhost:3000](http://localhost:3000)

## 📁 Project Structure

- `app/`: Contains the main application routes and layouts.
- `public/`: Houses high-resolution 3D assets and static files.
- `globals.css`: Defines our premium dark-mode design system.
